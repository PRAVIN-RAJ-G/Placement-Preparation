# ☕ Introduction to Java

## What is a Programming Language?

A programming language is a medium used to communicate with computers and perform specific tasks.

## Types of Programming Languages

Based on evolution, programming languages are classified into:

1. Low-Level Language
2. Medium-Level Language
3. High-Level Language

## High-Level Programming Language

A high-level programming language is human-readable and easy to understand.

**Examples:**
- C
- Java
- Python

---

# Java

Java is:

- A high-level programming language.
- A platform-independent programming language.

## Why is Java Platform Independent?

Java follows two steps during program execution:

1. Source code is compiled into **Bytecode** using the Java Compiler.
2. The **JVM (Java Virtual Machine)** executes the bytecode on any operating system.

```
Java Program
      │
      ▼
 Java Compiler
      │
      ▼
   Bytecode
      │
      ▼
      JVM
      │
      ▼
 Windows / Linux / macOS
```

---

## JDK (Java Development Kit)

JDK is used to develop Java applications.

- It is platform dependent.
- Different operating systems require different JDKs.

### JDK contains

- JRE
- JVM
- Development Tools (Compiler, Debugger, etc.)

---

## JRE (Java Runtime Environment)

JRE provides the environment required to run Java applications.

It contains:

- JVM
- Java Libraries

---

## JVM (Java Virtual Machine)

JVM executes Java bytecode.

It enables Java programs to run on different operating systems without changing the source code.

---

## Key Points

- Java is Object-Oriented.
- Java is Platform Independent.
- Java uses Bytecode.
- JVM makes Java portable.
- JDK is used for development.
- JRE is used for execution.
