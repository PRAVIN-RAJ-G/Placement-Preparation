# ☕ Introduction to Java

## What is a Programming Language?

A programming language is a medium used to communicate with a computer to perform a specific task.

---

## Types of Programming Languages

Based on evolution, programming languages are classified into three types:

1. Low-Level Programming Language
2. Medium-Level Programming Language
3. High-Level Programming Language

---

## High-Level Programming Language

A high-level programming language is a human-understandable programming language.

**Examples:**
- C
- Java
- Python

---

# What is Java?

Java is a high-level, object-oriented, and platform-independent programming language developed by Sun Microsystems in 1995.

---

## Features of Java

- High-Level Language
- Object-Oriented
- Platform Independent
- Secure
- Robust

---

## Why is Java Platform Independent?

Java follows a two-step execution process.

1. The Java source code is compiled into **Bytecode** using the Java Compiler.
2. The **JVM (Java Virtual Machine)** executes the bytecode on any operating system.

Because every operating system has its own JVM, the same Java program can run on different platforms.

**Write Once, Run Anywhere (WORA).**

---

## Java Execution Process

```
Java Source Code (.java)
          │
          ▼
      Java Compiler
          │
          ▼
      Bytecode (.class)
          │
          ▼
           JVM
          │
          ▼
     Operating System
```

---

# JDK (Java Development Kit)

JDK is a software development kit used to develop and run Java applications.

JDK is platform dependent because each operating system has its own JDK.

### Components of JDK

- JRE (Java Runtime Environment)
- JVM (Java Virtual Machine)

---

# JRE (Java Runtime Environment)

JRE provides the environment required to run Java applications.

It includes:

- JVM
- Required Java Libraries

---

# JVM (Java Virtual Machine)

JVM executes Java bytecode.

It converts bytecode into machine code and enables Java programs to run on different operating systems.

---
