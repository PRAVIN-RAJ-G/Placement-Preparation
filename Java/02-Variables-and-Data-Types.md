# ☕ Variables, Data and Data Types

## What is Data?

Data is the raw information about the state or behavior of an object.

### Types of Data (Values) in Java

There are five types of values in Java:

1. **Integer** – Numbers without decimal point.
   - Example: `1`, `25`, `100`

2. **Decimal** – Numbers with decimal point.
   - Example: `2.5`, `10.75`

3. **Character** – A single character enclosed within single quotes.
   - Example: `'A'`, `'1'`

4. **String** – A sequence of characters enclosed within double quotes.
   - Example: `"Java"`, `"Hello"`

5. **Boolean** – Represents either `true` or `false`.

---

# What is a Data Type?

A data type is used to create a memory location (container) to store data.

It decides what type of data can be stored inside the container.

---

## Types of Data Types

There are two types of data types:

1. Primitive Data Type
2. Non-Primitive Data Type

---

## Primitive Data Type

Primitive data types are used to store a **single value**.

There are **8 primitive data types**.

| Data Type | Description |
|-----------|-------------|
| byte | Integer value |
| short | Integer value |
| int | Integer value |
| long | Integer value |
| float | Decimal value |
| double | Decimal value |
| char | Single character |
| boolean | True or False |

---

## Non-Primitive Data Type

Non-primitive data types store data as **references**.

They are used to store multiple values or complex data.

### Examples

- String
- Array
- Class
- Interface

---

# What is a Variable?

A variable is the name given to a memory location (container) used to store data.

### Syntax

```java
dataType variableName = value;
```

### Example

```java
int age = 21;
String name = "Pravin";
double cgpa = 8.5;
```

---

# Rules for Declaring Variable Names

1. Variable names should not start with a number.
2. Spaces are not allowed in variable names.
3. Java keywords cannot be used as variable names.
4. Only `_` (underscore) and `$` symbols are allowed. Other special symbols are not allowed.


---

## Quick Revision

- **Data** → Raw information.
- **Data Type** → Specifies what type of data can be stored.
- **Primitive Data Type** → Stores a single value.
- **Non-Primitive Data Type** → Stores references to objects.
- **Variable** → Name of a memory location used to store data.
